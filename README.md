东方符斗祭
==========

简介
----------

东方符斗祭是一个以东方Project为背景的“三国杀”卡牌游戏的复刻。
使用了Python编写，使用pyglet（OpenGL）绘图。

游戏开发状况
----------

实现了24名人物，大部分原版卡牌以及少量原创卡牌。
实现了“休闲”3v3模式，以及标准身份场模式。
